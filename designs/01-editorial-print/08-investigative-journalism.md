# Design 08: Investigative Journalism Brief

**Category:** Editorial & Print-Inspired  
**Style Tags:** investigative, classified, evidence-board, redacted, exposé, manila-folder  
**Difficulty:** Advanced  
**Best For:** Competitive intelligence presentations, research deep-dives, problem investigation frameworks, and any presentation that reveals hidden patterns or uncomfortable truths. Ideal when the presenter wants to position themselves as a skilled investigator uncovering significant findings.

---

## Visual Direction

Investigative news magazine meets classified government document — the aesthetic of exposé layouts, redacted files, evidence boards, and connection maps. The design communicates systematic investigation, document trail analysis, and the revealing of things that were meant to remain hidden. Elements feel like they were assembled on an analyst's desk, not designed in a software tool.

## Background & Texture

Off-white aged paper or manila folder tan (#D4B483) as the base. Document scan artifacts — slight skew, edge shadows, coffee ring stains, staple holes, fold lines. Multiple document layers visible as if papers are stacked and overlapping. Faint security pattern background (the repeating "CONFIDENTIAL" pattern used on classified documents) at very low opacity in the background.

## Typography

Mixed typography simulating real documents: typewriter monospace (Courier) for document body text, stamped bold all-caps for classification labels, handwritten annotation style for investigator notes, redaction bars covering sensitive text. Headlines in bold newsprint condensed sans-serif. Handwritten red or blue annotations in the margins connecting elements.

## Color Palette

- **Primary:** Off-white / manila (#F5EDDA) — document background
- **Secondary:** Typewriter black (#1A1A1A) — primary text
- **Accent 1:** Classified red (#CC0000) — stamps, urgent annotations, critical findings
- **Accent 2:** Investigator blue (#1E3A8A) — handwritten notes, connecting lines
- **Neutral:** Manila tan (#C4A35A) — folder backgrounds, document separators

## Layout & Composition

Evidence board composition — items pinned at slight angles as if physically attached. Multiple overlapping document layers create depth. Red string or line connections between related documents. Classification stamps (CONFIDENTIAL, TOP SECRET, EYES ONLY) in bold red at document corners. Sticky note annotations in yellow. Documents can be slightly rotated (1–3 degrees) to feel physically placed rather than digitally aligned.

## Graphic Elements & Decorations

Black redaction bars covering certain text (even if the text is harmless — it signals investigation aesthetic). Red rubber stamp marks ("CLASSIFIED", "EXHIBIT A", "EVIDENCE"). String/line connections between related elements. Polaroid-style photo frames for evidence images. Paper clip graphics holding document groups together. Magnifying glass highlighting key evidence. Typewriter text with occasional strikethrough corrections.

## Slide Types & Templates

- **Title Slide:** Manila folder cover with typed investigation title, case number in top corner, classification stamp, file date in typewriter text.
- **Content Slide:** Multiple overlapping document excerpts with handwritten red annotations connecting key phrases, evidence classification numbers.
- **Process/Flow Slide:** Investigation timeline with dated events as pinned document cards, red string connecting the sequence, magnifying glass on the pivotal moment.
- **Data/Chart Slide:** Statistical evidence presented as typed tables with handwritten analyst notes circling key figures, a bold red finding callout.
- **Closing Slide:** "CONCLUSIONS" typed in all-caps Courier, bulleted findings with red checkmarks, case closure stamp, investigator signature.

## Tone & Mood

Revelatory, systematic, and urgent. The tone communicates: significant research has been done, patterns have been identified, and what has been found cannot be ignored. There is the intellectual thrill of discovery mixed with the seriousness of significant findings. Audiences lean forward.

## Avoid

- Clean, polished digital aesthetics
- Bright cheerful colors
- Rounded or card-based layouts
- Sans-serif only typography
- Photography that looks staged or stock
- Any element that would appear in a standard corporate presentation

## Example Prompt

> Create a presentation slide styled as an investigative journalism evidence board and classified document file. Use aged manila folder tan as the background with faint security pattern overlay, multiple slightly-rotated overlapping document layers with typewriter text, bold red rubber stamp marks ("CLASSIFIED", "EXHIBIT A"), handwritten red and blue annotation connections between key elements, black redaction bars over certain text sections, and a paper clip holding related documents together. Include a bold newsprint headline at the top. The feel should be revelatory, systematic, and urgent — like uncovering a significant story hiding in plain sight.
