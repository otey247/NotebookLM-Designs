# NotebookLM Design Styles — 100 Complete Design Specifications

A comprehensive catalog of 100 fully-specified presentation design styles for NotebookLM and Google Slides, organized into 10 categories. Each design includes visual direction, typography system, color palette, layout guidelines, graphic elements, slide templates, and a ready-to-use NotebookLM prompt.

---

## How to Use This Catalog

1. **Browse by category** to find designs that match your presentation's context and audience.
2. **Open a design file** to read the full specification.
3. **Copy the Example Prompt** at the end of each file and paste it into NotebookLM to generate slides in that style.
4. **Mix and match** — many elements from different designs can be combined.

## Structure of Each Design File

Each of the 100 design files follows this consistent structure:

```
# Design [N]: [Name]

**Category:** [Category Name]
**Style Tags:** [searchable tags]
**Difficulty:** Beginner / Intermediate / Advanced
**Best For:** [specific use cases]

## Visual Direction
## Background & Texture
## Typography
## Color Palette
## Layout & Composition
## Graphic Elements & Decorations
## Slide Types & Templates
## Tone & Mood
## Avoid
## Example Prompt
```

The **Example Prompt** section at the end of each file is designed to be pasted directly into NotebookLM to generate a presentation in that design style.

---

## Design Difficulty Levels

- **Beginner** — Clear, straightforward visual vocabulary. Works with default tools. 
- **Intermediate** — Requires deliberate design choices and some visual design knowledge.
- **Advanced** — Demands deep understanding of the reference tradition and sophisticated execution.
